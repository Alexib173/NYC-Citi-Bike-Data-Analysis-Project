# NYC-Citi-Bike-Data-Analysis-Project
This project analyzes the operational data of the NYC Citi Bike system to identify usage patterns, station imbalances, and user behaviors. The goal is to provide actionable insights for Lyft and the NYC Department of Transportation to improve efficiency and infrastructure planning.
## Files in this Repository
* The main Jupyter Notebook containing all data cleaning, transformation, and visualization code.
* 1-page summary of the findings and business recommendations.
* The final presentation deck highlighting key visuals.
* The raw dataset used for analysis

## Key Insights
* Casual riders take trips are more frecuent user than annual members.
* High-traffic transit hubs require significantly more aggressive rebalancing than residential docks.
* Weekend usage patterns differ drasticaly from weekday commuter spikes, requiring different staffing schedules.

## Tools Used
* Python (Pandas, Matplotlib, Seaborn)
* Jupyter Notebooks
* https://s3.amazonaws.com/tripdata/index.html
